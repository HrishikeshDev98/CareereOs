# CareerOS 🚀

A personal operating system to manage your entire job search journey.

## Purpose

This project is being built to strengthen React fundamentals by building a production-level application from scratch.

## Tech Stack

Frontend:
- React 19
- TypeScript
- Vite
- Shadcn UI
- React Router
- TanStack Query
- React Hook Form
- Zod

Backend:
- Supabase
- PostgreSQL
- Auth
- Storage
- Realtime
- Edge Functions

Deployment:
- Vercel
- Supabase
